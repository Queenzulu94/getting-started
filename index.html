Html Table Reference

Table Tag

Tag Name Description
table Table Creates a table element.
thead Table Head sets of rows that defines the column header.
tr Table Row element adds to a row in the table.
td Table Data adds data to a row in a table.
tbody Table Body set of row that contains actual data.

Table Attributes

Tag Name Description
colspan Column Span defines how many column a table data should span.
rowspan Row Span defines how many rows a table data can span.
